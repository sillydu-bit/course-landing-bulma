О курсе

Наш курс разработан для абсолютных новичков.

Вы освоите современные технологии: HTML, CSS, JavaScript, Git, React, Node.js и другие. Курс включает в себя:

  • Видеоуроки
  • Практические задания
  • Менторскую поддержку
  • Портфолио проектов

Программа курса

Модуль 1: Основы

HTML, CSS, Git

Модуль 2: JavaScript

Синтаксис, DOM, API

Модуль 3: Frontend

React, Redux, Стилизация

Модуль 4: Backend

Node.js, Express, Базы данных

Модуль 5: Финальный проект

Создание fullstack-приложения

Модуль 6: Портфолио & Карьера

Поиск работы, подготовка
